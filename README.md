# Twitter X Recreation
Next.js frontend on Vercel and Java backend on AWS.

## AWS Architecture

- Frontend: Next.js hosted on Vercel (SSR, API routes)
- Backend: Java (Spring Boot) on AWS (EC2)
- Communication: REST API between Vercel and AWS
